# Ask AI — What It Is and How to Use It

Dashboard section: (none — bottom-right chat bubble, "Ask AI" tab)

## What it is
Ask AI is the dashboard's built-in assistant, in the floating chat bubble at the
bottom-right (the bubble is draggable; the panel also has the separate Team Chat
tab). It answers questions about the MSFG Dashboard — what a feature does, where
a link or tool lives, and how to complete common tasks — using an approved set
of MSFG Dashboard guides. Answers show their Sources, and many include direct
links you can click to jump to the right place.

## How to get good answers
- **Use it like search.** Ask "where is X" or "how do I do Y" and the answer
  leads with the link or click path.
- **Name the feature in every question.** Each question is answered on its own —
  the assistant does not carry context between messages. A follow-up like "how
  do I use it" without naming the tool will miss; ask "how do I use Loan
  Sifter" instead.
- **One task per question** works better than several bundled together.
- "New conversation" starts a fresh thread; it does not change what the
  assistant knows.

## What it can NOT do
- **No live data.** It cannot look up current rates, loan or borrower records,
  pre-approval statuses, or any real-time numbers. It only knows what the
  guides describe; for live data it points you to the right dashboard section.
- **No personal employee details.** It cannot show a person's phone, email, or
  profile. To pull up a coworker's info, open the HR menu → Team Directory and
  click their name to see their Contact Card.
- **It does not learn from chat.** Telling it something in the conversation
  does not teach it. There is no in-chat teaching mode: new knowledge only
  arrives when an admin adds or updates a guide document in the Ask AI corpus.
  If an answer is wrong or a topic is missing, tell an admin (Zack) what you
  asked and what you expected — the fix is a corpus update.

## FAQ
**Q: Can Ask AI learn something if I explain it in the chat?**
A: No. The assistant answers only from the approved guides. Explaining a
workflow in chat does not store it. Ask an admin to add the workflow to the
Ask AI corpus — after the next corpus sync, the assistant can answer it.

**Q: Why did it say "I could not find that in the provided documents"?**
A: The topic isn't covered by any guide yet. That is the assistant refusing to
guess rather than making something up. Report the gap to an admin so the guide
can be added.

**Q: Why did a follow-up question get a confused answer?**
A: Each message is answered independently — the assistant does not see earlier
messages. Re-ask with the feature or person named in the question itself.
