Captured from https://dashboard.msfgco.com/ on 2026-07-22T16:04:27.645Z. This is the authenticated MSFG operational dashboard, separate from the suite LOS console but connected by user workflows and dashboard Ask AI.

## Purpose
The dashboard gives staff a single place for goals, news and announcements, pre-approvals, applications, loan pipeline, funded loans, investor/program lookup, HR links, marketing resources, calculators, document tools, scanner, and Ask AI/chat access.

## When to use it
Use it when staff need the current dashboard snapshot, a direct operational link, a pre-approval lookup, a pipeline/funded-loan summary, current announcements, or access to dashboard tools.
