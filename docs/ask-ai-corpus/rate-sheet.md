# Today's Rate Sheet

Dashboard section: (none — header toolbar button)

## What it is
The "Today's Rate Sheet" button in the header toolbar (top utility row) opens
the current MSFG rate sheet. It opens https://email.msfginfo.com in a new tab —
a viewer page that displays the latest published rate-sheet email.

## How to find it
Click the "Today's Rate Sheet" button in the dashboard's top header toolbar, or
go directly to https://email.msfginfo.com. There is also a "Rates" tab inside
the News/Announcements section for rate-related posts.

## Common tasks

### See today's rates
Header toolbar → "Today's Rate Sheet" → the viewer opens with the most recent
rate sheet.

## FAQ
**Q: How does the rate sheet get published or updated?**
A: The publishing workflow happens outside the dashboard and is not documented
in these guides yet — ask an admin (Zack) for the current process.
